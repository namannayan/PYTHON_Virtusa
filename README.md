# Smart Expense Tracker with Insights

## Overview

Smart Expense Tracker with Insights is a Python-based personal finance application designed to help users monitor daily spending, understand financial habits, and make better budgeting decisions.

The system stores expenses in a CSV file, generates visual reports, identifies major spending categories, and provides smart suggestions to reduce unnecessary expenses.

---

## Key Objectives

* Track daily expenses in a simple way
* Categorize spending habits
* Visualize where money is spent
* Detect overspending areas
* Encourage better financial planning

---

## Features

### Expense Management

* Add new expenses with:

  * Date
  * Category
  * Amount
  * Description

### Reports & Analytics

* Total Expense
* Average Expense
* Highest Single Expense
* Category-wise Expense Summary
* Highest Spending Category

### Data Visualization

* Pie Chart for expense distribution
* Bar Graph for category-wise spending

### Smart Suggestions

Provides useful recommendations such as:

* Reduce food delivery expenses
* Use public transport to lower travel costs
* Avoid impulse shopping
* Control utility bills
* Set a monthly budget if spending is high

### Data Control

* Reset all stored expense records
