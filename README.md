# Smart Expense Tracker with Paytm Insights

## Project Overview
This project analyzes Paytm transaction data using Python and provides useful spending insights with charts.

## Features
- Load Excel transaction data
- Clean and preprocess dataset
- Categorize expenses (Food, Travel, Bills, Shopping, etc.)
- Monthly spending insights
- Highest spending category and day
- Visualizations using Matplotlib & Seaborn
- Smart budget suggestions

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
